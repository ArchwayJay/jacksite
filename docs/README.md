Jack Johnson — Portfolio

This repository is a small portfolio site built with Flask for blog management and static pages for projects.

How to use

- Install dependencies and run the Flask app locally:

  python3 -m pip install -r requirements.txt
  python3 app.py

Then open http://127.0.0.1:5000

Admin

Use the admin panel at `/admin` to manage blog posts. This demo uses a simple session-based auth for convenience; replace with proper user management before production.
